 # 🔤 String Analyzer (C++)

A simple C++ program that:
- Reverses a string
- Counts the number of vowels
- Checks if the string is a palindrome

📌 Example

Input: level
Reversed string: level
Vowel count: 2
The string is a palindrome.

🛠️ Tech Stack
Language: C++
Concepts: Strings, Loops, STL, Conditions
